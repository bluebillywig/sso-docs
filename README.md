# SSO integration with Blue Billywig OVP
The Blue Billywig Online Video Platform (OVP) can be integrated with an external Identity Provider (IdP) using a SAML
2.0 connection.

Review the instructions for setting up an SSO connection for any one of the following Identity Providers:

* Integrating with [Okta](Okta.md).
* Integrating with [Microsoft Entra ID](EntraID.md).
* Integrating with [a generic SAML enabled Identity Provider](GenericSAML.md).
